Proyecto Curso de Investigación Pregradual

Bienvenidos a la presentación de mi proyecto de investigación. Este proyecto tiene como objetivo desarrollar una plataforma web para centralizar y consultar la información de programas de pregrado en Villavicencio, proporcionando una herramienta útil para estudiantes, padres y orientadores académicos.
